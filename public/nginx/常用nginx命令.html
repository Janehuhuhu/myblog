    # 常用 nginx 命令(mac系统)

    # 💡 常用命令

    1. 启动 nginx
    sudo nginx

    1. 重启 nginx
    sudo nginx -s reload

    1. 停止 nginx
    sudo nginx -s stop

    1. 检查 nginx 配置语法
      在启动 Nginx 服务之前，可以检查它的配置语法是否正确。尤其是当你修改或添加配置时很有用。命令如下：
    sudo nginx -t

    输出结果如下，即为配置正常

    nginx: the configuration file /etc/nginx/nginx.conf syntax is ok 
nginx: configuration file /etc/nginx/nginx.conf test is successful
